Enumeration Type Documentation

enum salad_outputfmt_t

The format specification for the model files used by salad.

Enumerator
SALAD_OUTPUTFMT_UNKNOWN 

Unspecified output format.

SALAD_OUTPUTFMT_TXT 

Human-readable, textual output.

SALAD_OUTPUTFMT_ARCHIVE 

Textual and binary output compressed as archive.

enum saladmodel_type_t

The type identifier for the model used by salad.

Enumerator
SALAD_MODEL_BLOOMFILTER 

Bloom filter as described by Burton H. Bloom in "Space/Time Trade-offs in Hash Coding with Allowable Errors" (1970), Communications of the ACM 13 (7): 422–426

SALAD_MODEL_NOTSPECIFIED 

An unspecified/ not initialized model.
